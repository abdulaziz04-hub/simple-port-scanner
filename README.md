# Simple Port Scanner

## Overview
This project is a basic TCP port scanner written in Python. 
It was built to understand socket programming and core networking concepts.

## Features
- Scans ports from 20 to 1024
- Detects open TCP ports
- Simple and beginner-friendly implementation

## How It Works
The script attempts to establish a TCP connection to each port in a specified range.
If the connection succeeds, the port is considered open.

## Skills Demonstrated
- Python fundamentals
- Socket programming
- Networking basics
